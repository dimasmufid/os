# Phase 3 — Expand the World (Depth & Exploration)

## Goal
Increase the sense of **place**, depth, and progression by expanding the world into a richer LifeOS environment.

---

# 1. Core Features

## 1.1 Additional Rooms
Examples:
- Library
- Garden
- Workshop
- Meditation Room
- Gallery of Achievements

Rooms map to different goals/habits.

## 1.2 Decorative Layering System
- Custom tiles appearing after certain milestones
- Seasonal decorations (optional)
- User-selected themes

## 1.3 First NPC
Basic NPC with:
- Idle animation
- Dialogue bubble
- “Tips” or motivational quotes
- Possible quests (future integration)

## 1.4 World Themes
Theme options:
- Classic plaza
- Night mode
- Cozy winter forest
- Japanese town
- Sci-fi lab

User can switch themes.

## 1.5 Avatar Customization Expansion
- Multiple color palettes
- New animation sets
- More cosmetic slots: Backpiece, Face Accessory, Shoes

---

# 2. Deliverables
- New tilemaps
- New themes (assets)
- NPC sprite & dialogue system
- Room expansions
- UI for world themes

---

# 3. Success Criteria
- User feels their world “grows” with them
- More exploration and meaning
- Stronger personal attachment
- Aesthetic diversity
